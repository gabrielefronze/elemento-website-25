Pricing Concepts

Predictable software cost. Transparent infrastructure cost.

Electros Subscription

Organisation-wide licensing for the Metacloud control plane.

  • Not per VM
  • Not per core
  • Not per socket
  • Not per resource
  • Not per seat
  • Not per user

Connection Credits

Connections activate integrations between Electros and infrastructure services.

Connection credits are included in each subscription tier and can be topped up.

AtomOS Endpoint

Connections to AtomOS infrastructure are included in all subscription tiers at no additional cost.

Atomosquare Third-Party Endpoints

VMware, Proxmox, OpenStack and other hypervisor environments consume one connection allocation per endpoint.

Atomosphere Cloud Service Classes

One service-class connection enables access to all supported providers implementing that service class.

One connection. All compatible providers.
